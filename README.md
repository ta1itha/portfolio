# Talitha O'Brien — Portfolio

Personal portfolio website targeting Associate Technical Product Manager roles.

**Live site:** https://ta1itha.github.io/portfolio

---

## About

A single-page portfolio built to tell a story, not just list jobs. Developer turned educator turned product thinker — each section reflects a version of the same person shipping value in a different context.

---

## Structure

```
index.html              # Single-page portfolio
assets/
  images/
    photo.jpg           # Profile photo
  docs/
    t_obrien.pdf        # CV / resume
  slides/
    w1/                 # BA Week 1 — Complaint Management CRM (8 slides)
    w2/                 # BA Week 2 — Retail Self-Service Checkout (10 slides)
    w3/                 # BA Week 3 — Hospital WhatsApp Scheduling (9 slides)
```

---

## Sections

- **Hero** — name, tagline, CV download
- **About** — career arc + stats (700+ students, 2 internships, 90% code reduction)
- **$ git log** — version-controlled work history (v0.0.1 → v1.3.0-beta)
- **Work Experience** — Stroop Digital PM internship case study + Miro board + BA internship carousels
- **Skills & Tools** — 4 categories: Product Management, Technical, AI & Automation, Tools & Platforms
- **Recommendations** — 6 LinkedIn recommendations with profile photos
- **Contact** — email + LinkedIn

---

## Tech

- Pure HTML, CSS, vanilla JS — no build step
- Google Fonts: Space Grotesk, Inter, Space Mono
- Neo-brutalist design (thick borders, offset box-shadows, hover lift)
- Fully responsive down to 375px
- Lazy-loaded images and deferred iframes for performance

---

## Deployment

Hosted on GitHub Pages from the `main` branch root.

To update: edit `index.html`, commit, and push. Changes go live automatically.
